# untocs Landing Page

Toxic-chemical-free baby clothing (0-2 years) - Waitlist landing page

## Features
- Clean minimal design with Nunito font
- Product showcase (10 baby clothing items)
- Waitlist form with Google Sheets integration
- Country code dropdown (34 countries)
- Fully responsive design

## Tech Stack
- React 19
- Tailwind CSS
- Shadcn UI
- Google Sheets API
